# Task Completion

Documentation slice is complete only when:

- Target module/layer, audience, trigger/end conditions, owner, evidence, related procedure, and legal-register references are present where applicable.
- Taiwan legal claims have an official source, verification date, effective-status check, and any population/size/capacity/contractor assumptions are explicit.
- Markdown headings, relative links, IDs, and table structure are internally consistent.
- Run `git diff --check`, inspect the actual focused `git diff`, and inspect `git status --short --branch`.
- Update and read back `mem:current_work_state` when a stage ends or work must continue later.
- Never report operational/legal compliance without evidence of required filing, training, qualification, inspection, risk assessment, permits, site controls, and records.
