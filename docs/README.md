## 權威邊界

- `governance/` 定義 canonical 模型、法源與 evidence。
- `enterprise/` 定義跨 HLC 的公司治理文件。
- `modules/` 定義每個 HLC 所需文件、gate、owner、evidence 與輸出。
- `shared/` 定義所有 Candidate artifact 共用的版本、簽署、核准與紀錄規則。
- 正式模板只能在公司 selector、owner、法律／專業審查及核准完成後建立。
