# HLC04 — 專案派任、配置與工地動員

> Relationship state：`active`
> Primary authorities：Resource／Delivery／Employer／Project／HSE authorities

## 模組目的

為每次專案、工作包或 Bench 轉入建立明確角色、配置比例、日常工作排序者、工作條件與 site gate；派任不建立另一個雇主。

## Gate 與輸出

- assignment 總配置比例、工時及跨案需求須通過 Resource Governance。
- 工作地點、班別、工資或職務重大改變時另開 HLC08，不以派任書單方處理。
- HLC04 只證明進場 gate；不取代作業許可、設備檢查或現場施工控制。
- 輸出為有效 assignment、project decision rights、site clearance 與 access set。
