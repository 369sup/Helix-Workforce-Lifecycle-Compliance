# HLC03 — 能力基線、公司職級與技術授權

> Relationship state：`active`
> Primary authorities：CoE／Technical Authority／HSE Authorization Authority

## 模組目的

把公司職級、專案角色、能力、法定資格與技術授權建成獨立物件，避免以職稱自動取得權限。

## 必要文件

| Artifact ID | 文件 | 類型／簽署 | Owner／Approver | 必要證據 |
| --- | --- | --- | --- | --- |
| HX-ART-HLC03-001 | Competency Matrix 與初始／定期能力評估 | register／authority-approval | CoE；Capability Authority | 能力等級、經驗、限制、評估與複核 |
| HX-ART-HLC03-002 | 專業職級評定 | record／authority-approval | CoE／HR；Grade Authority | 職級標準、評審、校準與生效日 |
| HX-ART-HLC03-003 | 技術／設計審查授權與簽核權限表 | authorization／authority-approval | Technical Authority；Authorization Authority | 範圍、依據、資格、期限、限制與代理 |
| HX-ART-HLC03-004 | 技術授權暫停、撤銷與回復紀錄 | record／delivery-receipt | Technical／HSE；Authorization Authority | 原因、影響工作、通知、補正與回復 gate |

## Gate 與輸出

- Training completion、competency、credential 與 authorization 不得互相自動推導。
- 授權與資格到期、工作改變或風險事件發生時重開本模組。
- 未授權工作必須停止或改由合格人員監督，不得以專案時程豁免。
- 輸出為可查詢的能力、職級、資格及授權狀態。
