# HLC00 — 人力規劃、職缺與招募

> Relationship states：`planning`、`candidate`
> Primary authorities：Business／Resource／Recruitment；Employer／Budget Approver

## 模組目的

在作出聘僱或合作承諾前，證明需求、預算、用工關係、職務、CoE 歸屬、工作地點、危害、資格與招募條件已被合理定義。

## Gate 與輸出

- 定期契約、派遣、承攬、顧問、外國人、未成年或實習安排必須進入相應 legal selector。
- 招募條件須有職務必要性；非必要隱私、保證金與證件留置不得出現在流程。
- 輸出為核准需求、職務版本、關係初評、招募資料規則及 candidate disposition。
- 進入 HLC01 不表示 Offer 已成立；未錄取案件須完成資料保存或刪除決定。
