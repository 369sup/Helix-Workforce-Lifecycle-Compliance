# HLC08 — 跨專案調動、借調及工作條件變更

> Relationship state：`active`
> Primary authorities：HR／Resource／Employer Authority

## 模組目的

對職務、地點、班別、工時、工資、雇主、臨時指派、借調或永久調動作 before／after 審查，並重算受影響控制。

## 必要文件

| Artifact ID | 文件 | 類型／簽署 | Owner／Approver | 必要證據 |
| --- | --- | --- | --- | --- |
| HX-ART-HLC08-001 | 調動／借調／工作變更評估 | checklist／authority-approval | HR／Resource；Employer Authority | 必要性、動機、不利益、勝任、距離協助、家庭、期間與回復 |
| HX-ART-HLC08-002 | 員工意見、條件差異與變更協議／通知 | record or agreement | HR；Employer Authority | before／after、員工意見、同意判斷、送達與 effective date |
| HX-ART-HLC08-003 | 原派任交接、權限異動與新 Assignment | work-item register／authority-approval | Resource／IT／Site；各 Authority | 交接、資格、安全、健康、保險、退休、資料與 access 重算 |

## Gate 與輸出

- Change type 至少分 temporary assignment、secondment、transfer、role、location、time、pay、employer 及 reclassification。
- 遵守 TW-WF-TRANSFER-001 的五原則，重大不利益不能只用 acknowledgement。
- 受影響的 HLC03、HLC04、HLC09 或 HLC11 gate 依差異重開。
- 輸出為核准變更、員工表示、新 assignment 及平行 remediation items。
